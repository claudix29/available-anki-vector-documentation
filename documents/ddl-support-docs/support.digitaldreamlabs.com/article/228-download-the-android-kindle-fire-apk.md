# Download the Android / Kindle Fire APK {#download-the-android-kindle-fire-apk .title}

[](javascript:window.print() "Print this article"){.printArticle
aria-label="Print this Article"}

**Note:** If you\'ve already downloaded the Cozmo app from the Google
Play or Amazon Appstore stores and are experiencing issues with the app,
this apk file will not fix those issues. This apk file is for customers
who are not able to download the app through the Google Play or the
Amazon Appstore stores but want to check if their device still might
work with Cozmo.

------------------------------------------------------------------------

::: sections
## [Requirements]{.wysiwyg-font-size-large}

For specifics about supported Android and Amazon Kindle devices please
refer to these articles:

[Cozmo Compatible
Devices](//support.digitaldreamlabs.com/article/135-compatible-devices){target="_blank"
rel="noopener noreferrer"}

For more details about device compatibility, for some suggestions on how
to potentially use your device, please refer to this article:

**[COZMO: Details about Device
Compatibility](//support.digitaldreamlabs.com/article/134-compatible-devices-further-information){target="_blank"
rel="noopener noreferrer"}**

### Please Note

If your device does not meet the requirements for using the Cozmo app or
has been intentionally excluded from installing it, the quality of your
experience using the Cozmo app may be problematic or it may not work at
all.
:::

------------------------------------------------------------------------

::: sections
## [[Warning]{.s1}]{.wysiwyg-font-size-large}

[In order to use this software, you must enable and keep enabled the
following setting:]{.s1}

[Settings \> More \> Security, then make sure ]{.s1}**[Unknown
sources]{.s2}**[ (Allow installation of applications from both trusted
and unknown sources.) is enabled (checked).]{.s1}

**This setting can potentially lower the overall security of your
device. Please consult the terms, policies and guidance from your device
manufacturer.**
:::
