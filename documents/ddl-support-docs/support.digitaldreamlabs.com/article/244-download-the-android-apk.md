# Download the Drive & OVERDRIVE Android APKs {#download-the-drive-overdrive-android-apks .title}

[](javascript:window.print() "Print this article"){.printArticle
aria-label="Print this Article"}

If you need a copy of the
[End-Of-Life](//support.digitaldreamlabs.com/article/254-anki-overdrive-end-of-life-eol){target="_blank"}
Anki DRIVE, OVERDRIVE, or OVERDRIVE: Fast & Furious Edition installation
packages for Android, you may download them here.\
\
If you\'ve already downloaded the OVERDRIVE app from the Google Play or
Amazon Appstore stores and are experiencing issues with the app, these
apk files will not fix those issues. These download apks are for
customers who are requesting an End-Of-Life app.

## Please Note

If your device does not meet the Minimum requirements or has been
intentionally excluded from installing OVERDRIVE / Fast & Furious
Edition apps, while you can still download and install the apk, you may
not be able to play single player games, you may not be able to host
multiplayer games, and the quality of your experience using these app
may be problematic or it may not work at all

You will only be able to play multiplayer games when someone with a
compatible Android device is hosting. To play multiplayer games, all
participants must be connected to the same Wi-Fi network and with some
routers on the same network channel.

## Installation Instructions

1.  Download the APK package for each app that you would like to
    install.
2.  Open your mobile device\'s file manager. In the Downloads folder (or
    wherever the app was saved after download), long press on the zipped
    package file name. A dialog box should appear.
3.  Click \"Unzip\" or \"Extract\" to unzip the APK package into a new
    folder.
4.  Navigate to the new folder. Click on the .apk file to open an
    installation dialog box. Click \"Install\". **Do not launch the app
    yet.**
    1.  If you receive a security error, please go into the Settings app
        of your mobile device.
    2.  Find the Security section.
    3.  Look for an option to be able to \"Install apps from unknown
        sources.\" Enable this option.
    4.  Go back to the beginning of Step 4 and reattempt installation.
5.  In your File Manager, navigate back to the folder with the APK you
    just installed.
6.  Use the Copy or Move function in your device to select and move/copy
    both of the \".obb\" files into the following directory in your
    phone\'s Internal Storage: \"Android/obb/\<app package name\>\"
    1.  For DRIVE, this will be \"Android/obb/com.anki.drive\"
    2.  For OVERDRIVE, this will be \"Android/obb/com.anki.overdrive\"
    3.  For OVERDRIVE: Fast & Furious Edition, this will be
        \"Android/obb/com.anki.foxtrot\"
7.  You may now launch the app and start the race!
