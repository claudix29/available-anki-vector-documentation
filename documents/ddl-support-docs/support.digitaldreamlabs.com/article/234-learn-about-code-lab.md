# Learn about Code Lab {#learn-about-code-lab .title}

[](javascript:window.print() "Print this article"){.printArticle
aria-label="Print this Article"}

In Cozmo\'s Code Lab, you can program Cozmo to perform any action you
want, using all of his sensors, motors, and speech abilities! 

![Cozmo_s_Code_Lab.png](//d33v4339jhl8k0.cloudfront.net/docs/assets/5e3f0b1e2c7d3a7e9ae777f5/images/5e7d77432c7d3a7e9ae9e3c9/img-3326-1585280661-1088952329.png)

::: sections
:::

------------------------------------------------------------------------

::: sections
## Specific questions about Cozmo\'s Code Lab?

For any questions about how to use Code Lab, build your own Projects, or
report issues, please take a look at the **[[Code Lab
Community]{.underline}](https://forums.anki.com/c/cozmo-code-lab){target="_blank"
rel="noopener noreferrer"}** forum located within the Cozmo SDK Forums.

Please note that this site requires you to Sign Up and create a
site-specific account. Your existing Anki Account will not work and
you'll need to be 13 years of age or older to access the community. If
you are not at least 13 years old, please get help from your parents to
sign up.
:::

------------------------------------------------------------------------

::: sections
## Code Lab is based on Scratch Blocks

Cozmo's Code Lab is based on Scratch Blocks, a collaboration between
Google and the MIT Scratch Team.
:::

------------------------------------------------------------------------

::: sections
## Requirements

-   iOS 9 or above
-   Android 5 (Lollipop) or above
-   Fire OS 5 or above

## Some Additional Notes 

###  

### **Sandbox Mode**

[After updating to Cozmo 2.1.0, Code Lab projects that you previously
created are now Sandbox projects.]{style="font-weight: 400;"}

[You can continue to use your Sandbox projects, create new ones from
scratch, or rename **[Sample
Projects](https://www.anki.com/cozmo/code-lab/sample-projects){target="_blank"
rel="noopener"}** as Remixes to jump start your
projects.]{style="font-weight: 400;"}

[As of Cozmo 2.2.0, you can now make copies of Sandbox Projects just
like Constructor Projects. Please take a look at our **[Code Lab
FAQ](https://www.anki.com/cozmo/code-lab-faq){target="_blank"
rel="noopener"}** if you have questions about how this process
works.]{style="font-weight: 400;"}

[For more information about Sandbox Mode, please click
**[here](https://www.anki.com/cozmo/code-lab/how-it-works){target="_blank"
rel="noopener"}**.]{style="font-weight: 400;"}

### **Constructor Mode**

[Constructor Mode is an all new way to take your coding skills to the
next level. For more details on what Constructor Mode has to offer and
how it compares to Sandbox Mode, please visit **[the new Constructor
Mode
page](https://www.anki.com/cozmo/code-lab/constructor-mode){target="_blank"
rel="noopener"}**.]{style="font-weight: 400;"}

[Just like Sandbox Mode, Constructor Mode comes with some
amazing **[Sample
Projects](https://www.anki.com/cozmo/code-lab/sample-projects){target="_blank"
rel="noopener"}** from which you can create Remixes to jump start your
projects. Constructor Mode adds the ability to save your projects to
Google Drive and **[submit them to
Anki](https://www.anki.com/cozmo/code-lab-faq){target="_blank"
rel="noopener"}** for a chance to be featured on Anki\'s
site.]{style="font-weight: 400;"}

[To start using Google Drive with your Constructor projects, download
the Google Drive app from the Apple App Store or the Google Play store.
For more details about using Google Drive with Constructor Mode, please
take a look at our **[Code Lab
FAQ](https://www.anki.com/cozmo/code-lab-faq){target="_blank"
rel="noopener"}**. ]{style="font-weight: 400;"}

[For Kindles (which do not support Google Drive), you can use email to
backup your Constructor projects. Please take a look at our **[Code Lab
FAQ](https://www.anki.com/cozmo/code-lab-faq){target="_blank"
rel="noopener"}**.]{style="font-weight: 400;"}

[Google Drive and email are each great ways to make copies of your
Constructor projects just in case you have to reinstall the Cozmo app or
if you want to move them to another device. Making copies just in case
or to save your progress with a particular Constructor project is always
a good idea.]{style="font-weight: 400;"}

[If you'd like Anki to consider featuring your Constructor projects,
please visit
**[go.anki.com/submission](https://go.anki.com/submission){target="_blank"
rel="noopener"}**. You'll need to create or login with your existing
Anki Account (you might have created one when you purchased Cozmo from
Anki). For details about an Anki Account, please visit
**[here](https://www.anki.com/account){target="_blank"
rel="noopener"}**. ]{style="font-weight: 400;"}[Please note that
Canadian and Japanese customers cannot currently create an Anki
Account.]{style="font-weight: 400;"}

For a more advanced programming experience with Cozmo, please take a
look at the [**Cozmo
SDK**](https://support.anki.com/hc/articles/222151647){target="_blank"
rel="noopener noreferrer"}.
