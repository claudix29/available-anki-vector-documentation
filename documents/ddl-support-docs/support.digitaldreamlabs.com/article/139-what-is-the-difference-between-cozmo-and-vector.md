# What is the difference between Cozmo and Vector? {#what-is-the-difference-between-cozmo-and-vector .title}

[](javascript:window.print() "Print this article"){.printArticle
aria-label="Print this Article"}

![](//d33v4339jhl8k0.cloudfront.net/docs/assets/5e3f0b1e2c7d3a7e9ae777f5/images/5e85629c2c7d3a7e9aea3b18/file-hY3MtoDAen.png)

Vector and Cozmo both have a place in the home, but they have different
ways to interact with you and your household.

+-----------------------------------+-----------------------------------+
| Cozmo                             | Vector                            |
+-----------------------------------+-----------------------------------+
| With a big brain and even bigger  | The home robot that hangs out and |
| personality, Cozmo is the         | helps out! \                      |
| ultimate robot for kids to play   | Ages: 18+                         |
| and learn with. \                 |                                   |
| Ages: 8+                          |                                   |
+-----------------------------------+-----------------------------------+
| Buy Cozmo if... \                 | Buy Vector if... \                |
| \                                 | \                                 |
|                                   |                                   |
| -   You want a toy                | -   You want to a robot to        |
| -   You\'re about having fun and  |     respond to your commands      |
|     learning to code in short     |     (instead of app commands)     |
|     play sessions                 | -   You want your robot to help   |
| -   You\'re looking for a STEM    |     you around the house, like    |
|     (Science Technology           |     set timers and check the      |
|     Engineering and Mathematics)  |     weather                       |
|     and Education product         | -   You want to get football      |
| -   Your robot is focused around  |     scores                        |
|     games, nurturing, unlocking   | -   You want your robot to be     |
|     skills, and learning to code  |     able to find his way back to  |
|                                   |     his Charger, always powered   |
|                                   |     on and connected              |
+-----------------------------------+-----------------------------------+
| Key Differences: Cozmo \          | Key Differences: Vector \         |
| \                                 | \                                 |
|                                   |                                   |
| -   App-controlled via a          | -   Voice command control, your   |
|     compatible Android, Fire OS,  |     compatible smart device just  |
|     or iOS smart device           |     needed for setup              |
| -   Education applications        | -   Autonomous and self-charging  |
|     including SCRATCH programming | -   Cloud-connected               |
|     via Anki\'s Code Lab          | -   1 Cube                        |
| -   Interactive games             | -   Can take pictures at your     |
| -   3 Cubes                       |     command                       |
+-----------------------------------+-----------------------------------+
| Included in the Box: \            | Included in the Box: \            |
| \                                 | \                                 |
|                                   |                                   |
| -   Cozmo Robot                   | -   Vector Robot                  |
| -   Charging platform             | -   Vector Charger                |
| -   3 Cubes                       | -   1 Cube                        |
| -   Quick Start Guide             | -   Quick Start Guide             |
| -   Product Information Guide     | -   Product Information Guide     |
| -   Cozmo Sticker                 |                                   |
+-----------------------------------+-----------------------------------+
| Required / Not Included: \        | Required / Not Included: \        |
| \                                 | \                                 |
|                                   |                                   |
| -   A current Android, Fire OS,   | -   A current Android, Fire OS,   |
|     or iOS smart device (see      |     or iOS smart device (see      |
|     compatible device list)       |     compatible device list)       |

|     Output = 5V/1.5A)             |     Output = 5V/1A)               |
|                                   | -   A validated, free Anki        |
|                                   |     Account                       |
